// Pattern: Edge-Based Maximum Distance

// When problem asks for MAX distance:
// → Always try using array edges (0 and n-1)

// Why?
// Because max distance = farthest points

// Strategy:
// 1. Compare all with first element
// 2. Compare all with last element
// 3. Take max

// Time: O(n)

// Use Cases:
// - Max distance problems
// - Greedy observation based questions
